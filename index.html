Sugestão Word

O Word é um software de processamento de texto bastante utilizado para criar e editar documentos. É uma ferramenta indispensável tanto para estudantes quanto para profissionais na criação de documentos, cartas, relatórios, artigos, entre outros. Com essa ferramenta, é possível formatar o texto de diversas maneiras, utilizando diferentes fontes, tamanhos, cores e espaçamentos, além de incluir imagens, tabelas e gráficos.

A seguir, assista aos vídeos e conheça as principais funcionalidades.
